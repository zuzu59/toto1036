# z-PWA Contacts

Petit POC de carnet de contacts offline-first, installable comme PWA.

## Stack

- Vite
- Vue 3
- TypeScript
- Dexie.js
- vite-plugin-pwa

## Fonctionnalités

- création de contacts
- consultation / modification
- suppression
- recherche locale
- export / import JSON
- export / import CSV
- fonctionnement hors ligne
- lien vers les releases GitHub depuis l’app

## Lancer le projet

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
